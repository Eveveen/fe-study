## js实现函数重载    
- 通过判断 arguments.length 实现   
```
    function fn() {   
        switch(arguments.length) {   
            case 0:   
                return '没有参数';   
            case 1:   
                return arguments[0] + 10;   
            case 2:   
                return arguments[0] * arguments[1];   
            default:   
                return;   
        }    
    }   
    fn();       // 没有参数   
    fn(10);     // 20   
    fn(5, 10);  // 50   
```
