## toPrecision 和 toFixed 和 Math.round 的区别   
- toPrecision 处理精度，从左至右第一个不为0的数起
```js
let a = 123.45;
let b = a.toPrecision(4); // 123.4
```
   
- toFixed 是对小数点后指定位数取整，从小数点开始数起。
```js
let a = 123.45;
let b = a.toFixed(1); // 123.4
```
   
- Math.round 是将一个数字四舍五入到一个整数。
```js
let a = 123.45;
let b = Math.round(a); // 123
```
